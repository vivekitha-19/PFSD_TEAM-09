The system uses:
  - facebook/bart-large-mnli → BERT zero-shot classification
  - mistralai/Mistral-7B-Instruct → advisory generation
  
When you submit a query:
  1. BERT classifies the crop stress type (real AI, not keywords)
  2. Mistral-7B generates a fresh advisory response
  3. Response is stored in MongoDB Atlas

TRAINING DATA — HOW TO MAKE MODEL BETTER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The "Retrain AI Model" button in the dashboard does NOT require you
to manually edit CSV. But if you WANT to add more data:

1. Open:  ml_model/training_data.csv
2. Add rows in format:  text,label
   Example:
     rice plant leaves getting yellow,nutrient_deficiency
     white powder on wheat leaves,fungal_disease
3. Click "Retrain AI Model" button on dashboard
   OR run: cd ml_model && python train_model.py

Labels allowed:
  nutrient_deficiency, fungal_disease, water_stress,
  pest_infestation, viral_disease, bacterial_disease,
  heat_stress, weed_infestation

Current: 415 rows → 66.3% accuracy
Target:  500+ rows per class → 90%+ accuracy
